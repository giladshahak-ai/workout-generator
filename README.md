# מחולל האימונים שלי 💪

אפליקציה אישית (קובץ HTML יחיד, ללא התקנה) לבניית אימוני כושר פונקציונליים של ~50 דקות —
כולל חימום, גוף אימון ומתיחות — בלחיצת כפתור.

## יכולות
- **✨ צור** — מחולל אימונים אוטומטי לפי קריטריונים: מיקוד (גוף מלא / עליון / תחתון / בטן / אירובי),
  אופי (מאוזן / אירובי / כוח), פורמט, וציוד זמין.
- **🛠️ בנה** — בניית אימון ידנית: בחירת פורמט לכל בלוק, הוספת תרגילים מתוך מאגר של 70 תרגילים,
  ותצוגה מקדימה חיה.
- **💾 שמורים** — שמירת אימונים על המכשיר (localStorage).
- **📚 מאגר** — 31 אימונים היסטוריים שנאספו לאורך השנים.
- **🧠 פורמטים** — הסבר על 7 פורמטים: טבטה, EMOM, AMRAP, תחנות, צבירה, פירמידה, סופר-סט.
- **⏱️ טיימר** — טיימר אינטרוולים עם צליל ורטט.
- **🌙/☀️ ערכת צבעים** — מצב כהה או בהיר-פסטל (נשמר אוטומטית).

## שימוש
פותחים את הקובץ `מחולל-אימונים.html` בדפדפן (מחשב או נייד) — זהו. אין שרת, אין התקנה.
